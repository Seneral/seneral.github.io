Getting Started

  1. Get TerrainComposer 2 from the Asset Store, download and import it.
  2. Get TC2 Node Painter from the Asset Store, download and import it.
  3. Create a new TC2 project through the TC2 window ('Window/Power of Nature Software/Terrain Composer 2')
  4. Add a node to the graph for painting, eg. in the height output - it will automatically be turned to an apropriate Image node in the next step.
  5. Add the NodePainter script to that Node. You will be presented with the TC2 Node Painter UI
  6. Create a new canvas by specifying a desired resolution and start painting!

Interface Explanation

Canvas IO

This section is for creating, loading and saving the canvas.
When there is no canvas loaded, you can create a canvas using the GUI in the outlined box, specifying size and color/grayscale format.
Canvas IO works for both raw and png image formats. To load a png file, drag it into the 'Import' object field, and save it using the button next to it. Raw files can be handled in up to 16Bit using the 'Import Raw' and 'Export Raw' buttons below.
Under this region are two Undo buttons.

Preview

Here you can get a preview of the canvas and also paint directly on it using the GUI. You can specify borders around the preview in the settings to aid painting the edges.
At the right to the header you can edit the canvas name and see the painter's ID by clicking on the 'E' button. You can also switch between color and grayscale modes after creation by clicking on the rightmost button.
When in color mode, you can select which channels to show in the preview from the toolbar below the canvas preview.
Due to the size of the canvas preview you have yet another two undo buttons below this region.

Color

Only shown when in Color mode, this region allows you to edit the color and it's intensity and save it to a preset bar next to it's header.

Multi Channels

Only shown when in Multi mode, you can select and modify the individual channels here.

Brush

This region is for editing the current brush and once again saving it to the numerated brush preset bar next to the header.
You can choose between various brush shapes here, the first one being an adjustable function, the others being 512px brush textures. The mode dropdown let's you choose between different blending modes of the brush. For a detailed overview on these, see 'Blend Modes Explanation' below. Furthermore you can adjust size, intensity and additional mode-specific options like Target and Bias here. Finally, a clamp option allows you to clamp your input to 0-1.

Modifications

This regions applies post-effects onto the canvas, not affecting the currently saved canvas until applied. When any modification is active, a warning symbol appears next to the header.
You can adjust the overall contrast and brightness, aswell as those of individual channels when in color mode. As it is a post-effect until explicitly applied to the canvas, you can reset and apply them using the apropriate buttons.
Additionally, not being a post-effect modification, you can set the canvas to an uniforma value/color in this region.

Bottom

In the 'Resize' region you can resize your canvas here, the content is apropriately up- or downscaled.
In the 'Node Targets' list below you can specify to which nodes the canvas is automatically applied to.
Pressing the 'Show Settings' button at the very bottom opens up the settings window for TC2 Node Painter.

Canvas Formats

The term 'canvas format' has to be understood in an untraditional sense. Internally, the canvas is always RGBA.
Instead, the canvas format specifies how you can use the canvas and for what purpose.
These are the three formats you have to choose from when creating a canvas:

Grayscale

A grayscale canvas is useful for height painting and simple masks in any output.

Color

A color canvas is not directly used in any output but is there for various reasons. It can be used for applications outside of TC2 or to get interesting results when mixing and blending the individual channels together (TC2 interprets them as max(r,g,b,a)). The color region is only shown here and you also get per-channel modifications.

Multi-Grayscale

This complex format allows you to paint grayscale in multiple channels, similar to splat painting - except that it is not only used for splats, but for e.g. multiple different biomes or forests.
This canvas format is also potentially consisting of multiple textures, each of the channels corresponds to one channel on the textures - that means, the painter uses one texture each 4 channels.
Using this format shows the Multi Channel region, where you can select and edit a channel. You can also choose to normalize the channel values in the brush region, which is strongly recommended because overlapping channels will fight, and the channel which is the highest in TC2 wins, which usually gives unwanted results. But this also dramatically slows down painting when using 8 or more channels.

Blend Modes Explanation

There are 4 major blend modes available:

  • Add / Sustract:
    This is a basic addition (or subsraction when holding shift) of the intensity onto the canvas.
  • Lerp:
    This mode lerps the canvas to the specified target, getting smoother the closer it is to the target (in contrast to Replace). Intensity sets the speed of the lerp. This can also be used as a smoother add brush and can be inverted by holding shift (sets 1-target).
  • Replace:
    Linearly replaces the canvas value with the target value at the speed specified by the intensity. This mode can also be inverted by holding shift (sets 1-target).
  • Smoothen / Contrast:
    Decrease or Increase details on the canvas texture with detail level specified through the bias property (shift to switch in between).

Color/Brush Presets

The preset system is available to both color and brush regions. You can load a preset by clicking on it's button, even it it's collapsed state, and create one by clicking the plus button. You delete them by right-clicking and selecting 'Delete' from the shown contect menu.
These presets are saved to your Editor
